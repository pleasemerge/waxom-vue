# Waxom-vue
Company website template

Demo: [https://pleasemerge.github.io/waxom/](https://pleasemerge.github.io/waxom-vue)

- Vue 3
- TailwindCSS
- Vuex

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
